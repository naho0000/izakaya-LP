# iPhoneでの確認方法

## GitHub Pages公開後

1. iPhoneでSafariを開く
2. GitHub PagesのURLを開く
3. 画面下の共有ボタンを押す
4. 「ホーム画面に追加」を選ぶ
5. 名前を「ヒロノブ」にして追加

## 見え方の確認ポイント

- 文字が小さすぎないか
- 下部ナビがホームバーに被っていないか
- 予約フォームの入力欄が押しやすいか
- カレンダーアイコンを押すと日付選択が展開するか
- 画像表示が重すぎないか

## 注意

GitHub Pages反映には数十秒から数分かかることがあります。
古い表示が残る場合は、Safariの再読み込み、または履歴とWebサイトデータの削除をしてください。
